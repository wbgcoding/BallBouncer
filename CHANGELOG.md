# Changelog

All notable changes to Ball Bouncer are documented here.

## [1.3.0] - 2026-05-17

- i18n support across 8 languages
- Animation speed control (global playback multiplier)
- Cross hole cycle button — cycles the gap through each arm
- Dynamic hole size slider
- New cross shapes row added to picker
- 3D portal dots on cross vertices
- Rainbow ball icon with arm cycling
- New themes: SciFi, Inferno, Vibrant, Aurora
- Direction label simplification (L/R toggles)
- Performance optimizations for smooth 60 fps

## [1.2.0]

- UI rework with improved layout and panel styling
- 32 outer shapes (up from 16)
- Color deduplication across themes
- Favicon added
- Random theme cycling — auto-rotates all three theme layers
- Random background cycling
- Ball color theme rows (including techy colors in row 2)
- L/R direction toggles for outer shape and cross
- Cross hole size and cycle sliders
- Cross hole cycles through arms over time
- Logo arm cycling animation
- Portal dots on cross vertices

## [1.1.0]

- Cross hole feature — configurable gap in the inner cross that balls can pass through

## [1.0.0]

- Initial release — balls bouncing inside a spinning geometric shape with a configurable inner cross
