# ⬡ Ball Bouncer

**Interactive physics playground** — launch balls inside spinning geometric shapes and watch the chaos unfold.

> Live demo: [ball-bouncer.vercel.app](https://ball-bouncer.vercel.app)

---

## Features

- **32 outer shapes** — polygons (triangle → 16-gon), stars, and hand-crafted characters: Ghost, Cat, Pac-Man, Heart, Crown, Skull, Fish, Rocket, Bat, Bell, Cloud, Flame…
- **18 inner cross shapes** — plus, X, Y, 5–12 arm crosses, Snowflake, Pinwheel, Star-Cross
- **14 color themes** — full color-wheel coverage: Blood, Lava, Ember, Gold, Toxic, Forest, Mint, Cyber, Ice, Ocean, Violet, Galaxy, Neon, Sakura
- **14 ball palettes** — Fire, Copper, Acid, Aurora, Ice, Cosmos, Neon, Candy, Pastel, and more
- **14 background fog modes** — static tints, cycling rainbow, fast fireworks
- **Shape size & speed** sliders for both the outer shape and inner cross
- **Comet trails** — bezier-curve trails with glow
- **Spatial audio** — pitch-shifted bounce sounds with reverb, panning by position
- **Random Themes** button — auto-rotates all three theme layers every 20–40 seconds
- **Responsive** — fills any window size

---

## Controls

| Panel section | What it does |
|---|---|
| Shape | Pick outer shape, set size & rotation speed |
| Cross | Pick inner cross shape, set size & rotation speed |
| Ball Colors | Ball color palette |
| Balls / Size / Speed / Gravity | Physics sliders |
| Random Size | Variance in ball radius |
| Fog | Comet trail length |
| Background | Fog tint + fade persistence |
| Theme | Shape/panel color theme |
| Random Themes | Toggle auto-rotating themes |
